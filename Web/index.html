What is Cyrillic URL Detector?

This browser extension protects you against sophisticated phishing attacks that exploit Unicode characters. Attackers often create fake URLs by replacing Latin letters with visually identical Cyrillic characters. For example, the Latin 'a' (U+0061) looks identical to the Cyrillic '–∞' (U+0430), but they're different characters that lead to different websites.

How it Works

When you visit a webpage containing Cyrillic characters in its URL, you'll immediately see this warning banner:

For example, these URLs might look identical, but one uses Cyrillic characters:

Safe: example.com (Using standard Latin 'e')
Dangerous: –µxample.com (Using Cyrillic '–µ' - a completely different character)

While some Cyrillic URLs are legitimate (like Russian websites), be extra cautious when seeing Cyrillic characters in URLs of international websites or financial institutions.

Technical Details

The extension works by:

  • Monitoring the current page URL
  • Checking for characters in the Unicode range U+0400 to U+04FF (Cyrillic script)
  • Properly handling URL-encoded characters
  • Injecting a warning banner when Cyrillic characters are detected
